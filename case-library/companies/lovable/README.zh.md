# Lovable

> Lovable 的 AI 产品增长连接了信任、自然分发、产品发布、付费获客时机和商业化。

**[English](README.md)｜[简体中文](README.zh.md)**

## 公司概览

Lovable 适合用于研究 AI 原生软件增长。在这个领域，产品能力快速变化，公开演示本身也可能成为分发。它涉及信任、创始人与员工公开参与、自然渠道、产品发布、付费获客时机和使用成本等问题。
