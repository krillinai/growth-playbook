# Eventbrite

> 一个有限的产品组织案例，聚焦可持续获客、CPO 领导力、增长所有权，以及产品结构如何适应变化的市场环境。

**[English](README.md)｜[简体中文](README.zh.md)**

## 背景

Casey Winters 曾担任 Eventbrite Chief Product Officer，管理产品、设计、研究与增长营销。他使用这段经历讨论产品领导力、增长战略、组织沟通与标准框架的边界。

现有证据更适合支持运作判断，而不是具体 Eventbrite 实验或公司结果的因果结论。

## 可持续获客

Winters 区分早期“Kindle”工作与可规模化“Fire”策略。人工获客可以帮助发现留存分群，只有在产品、客户与机制得到足够理解后，才适合加入规模化渠道。

对于活动 Marketplace，供需会随地区、品类、日期、主办方质量与活动频率变化。总流量或活动数量不能证明主办方和参与者获得持续价值。

## 产品与增长领导力

CPO 内容强调：

- 在决策会议前沟通战略与取舍；
- 让管理层理解不显眼的产品改进为何重要；
- 定义产品、增长、设计、研究与营销的所有权；
- 在保护已有价值的同时判断哪些方向值得继续规模化；
- 避免用框架掩盖真实约束。

这些属于组织原则，不能证明某一种结构造成 Eventbrite 的结果。

## 增长边界

增长团队需要掌握相关杠杆，但广泛所有权会增加维护与协调成本。产品负责人应定义结果、决策权、职能接口，以及临时增长问题何时转化成稳定产品所有权。

市场冲击和品类结构变化也会限制优化空间。更好的执行不一定能抵消活动环境或商业模式约束的变化。
