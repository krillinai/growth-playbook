# Pinterest

> 一个围绕 SEO（搜索引擎优化）入口页、转化、新用户引导、留存、国际增长与增长团队所有权演进展开的消费产品增长案例。

**[English](README.md)｜[简体中文](README.zh.md)**

## 增长背景

用户可以通过单个 Pin、Board 与 Topic 页面发现 Pinterest，因此大量潜在用户从未登录内容页进入，而不是从首页进入。增长工作需要把搜索意图和内容价值连接到产品理解、注册、激活与留存。

Casey Winters 的演讲来自他在 Pinterest 的增长与产品工作，属于参与者证据，不构成完整公司历史或因果拆解。

## 把 SEO 建成结构化页面系统

Winters 描述了三类创建搜索 Landing Page 的方法：

1. 使用结构化 Marketplace 或产品供给；
2. 使用 UGC（用户生成内容）；
3. 当供给和 UGC 不足时，主动生产有用内容。

Pinterest 可以围绕意图或主题聚合 Pin，形成可索引页面：

```text
搜索意图
-> 相关结构化内容
-> 有用的未登录页面
-> 产品理解
-> 合格注册
-> 激活与留存
```

页面数量和搜索流量只是中间指标。重复、单薄、误导或低意图页面会伤害用户与搜索表现。

## 优化真实入口界面

只优化首页无法覆盖从 Pin 和 Topic 页面进入的用户。Winters 描述了如何在高流量入口页测试注册提示，并简化 Modal 体验。

更通用的方法是按真实入口与意图绘制流量。转化率需要与理解度、用户满意、激活、留存和搜索质量护栏一起阅读。

## 解释产品广度

定性研究显示，一些访问者能理解眼前具体内容，却无法推断 Pinterest 更广泛的用途。因此，产品教育也是转化的一部分：在要求用户投入前，先让他们理解足够的价值模型。

这里存在取舍。强注册墙可能提高即时注册，更开放的探索则可能改善理解与信任。正确设计取决于下游留存价值。

## 新用户引导与留存

Setup-Aha-Habit 模型可以直接应用：

| 阶段 | Pinterest 的问题 |
| --- | --- |
| Setup | 需要哪些兴趣、主题或初始信号？ |
| Aha | 用户是否看到个人相关灵感，并理解 Pinterest 的用途？ |
| Habit | 收藏或推荐内容是否创造回访理由？ |

引导完成率不能单独验证模型。关键在于新 Cohort（同期群）是否形成留存且承载价值的行为。

## 国际增长

国际增长不只是翻译首页。不同市场的搜索供给、主题相关性、内容可用性、入口页面、新用户引导和产品理解都可能不同。

这个案例支持按市场诊断和投资内容质量，但没有公开完整国际扩张模型。

## 证据与局限

- 公开演讲简化了内部实验、分群、搜索系统与组织决策。
- 相对提升数字缺少完整基线、护栏、曝光细节与长期反事实。
- Pinterest 的结果还来自内容供给、产品质量、品牌、竞争、搜索引擎、众多团队与市场时机。
- 历史 SEO 与注册墙方法需要按照当前搜索政策和用户预期重新评估。
