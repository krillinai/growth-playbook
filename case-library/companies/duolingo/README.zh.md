# Duolingo

> Duolingo 的消费增长连接自然获客、国际化、品牌声音、订阅、实验、留存、召回与游戏化。

**[English](README.md)｜[简体中文](README.zh.md)**

## 公司概览

Duolingo 是采用 Freemium 模式并提供订阅服务的消费学习产品。它适合用于研究产品习惯、品牌、国际覆盖、商业化和实验如何相互加强。

当前内容结合 Gina Gotthilf 对自然增长、品牌、订阅杠杆和国际化的叙述，以及 Albert Cheng 关于 Explore-Exploit、留存、实验和游戏化的跨公司框架。

## 证据与局限

- 当前证据主要依赖参与者访谈，而不是完整内部实验和 Cohort 数据。
- Duolingo 的公开增长不能归因于品牌、游戏化、单独人物或单一机制。
- 这些方法对应高频消费学习产品，不一定适用于低频或企业产品。
- 收录不代表认同公司的全部决策、营销活动或产品主张。
