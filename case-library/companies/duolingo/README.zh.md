# Duolingo

> Duolingo 的消费增长连接自然获客、国际化、品牌声音、订阅、实验、留存、召回与游戏化。

**[English](README.md)｜[简体中文](README.zh.md)**

## 公司概览

Duolingo 是采用 Freemium 模式并提供订阅服务的消费学习产品。它适合用于研究产品习惯、品牌、国际覆盖、商业化和实验如何相互加强。

当前内容结合 Luis von Ahn 关于使命、Freemium、动机、品牌、文化与 AI 的创始人视角，Gina Gotthilf 对自然增长、订阅和国际化的叙述，Albert Cheng 的跨公司增长框架，以及 Jackson Shuttleworth 对 Streak 与留存的实践。

## 公司案例：Duolingo：消费订阅增长

### 案例速览

| 字段 | 信息 |
| --- | --- |
| 公司 | Duolingo |
| 覆盖时期 | 2023 年与 2025 年访谈讨论的实践 |
| 增长问题 | 把广泛免费使用转化成持续学习参与和订阅价值 |
| 主要机制 | 自然获客、国际化、品牌、订阅杠杆、实验、留存、召回与游戏化 |
| 参与者视角 | Luis von Ahn、Gina Gotthilf、Albert Cheng、Jackson Shuttleworth |

### 案例摘要

Duolingo 的订阅系统建立在 Freemium 学习产品之上。广泛可用性和国际覆盖创造获客，产品习惯与可识别的沟通促进用户回来，订阅体系再把部分使用转化成收入。

这套系统不能简化成吉祥物、连续记录、Paywall 或热门社交账号。现有来源呈现的是产品、品牌、市场与商业化机制组合，每一部分都需要单独衡量。

```text
可获得的产品价值
→ 自然与国际化触达
→ 习惯与品牌识别
→ 订阅转化
→ 留存与召回
→ 实验与再投入
```

### 1. 拆解订阅增长

Gotthilf 从多个方面理解 B2C 订阅增长，而不是只看一个转化率。实用模型可以拆成：

```text
合格获客
× 激活
× 付费转化
× 订阅留存
× 每位留存订阅者收入
```

这样可以避免 Paywall 的局部提升掩盖学习参与变弱、获客质量下降或后续取消。

### 2. 使用自然分发，但不混淆触达与价值

Duolingo 的产品文案、提醒和社交参与提供了容易识别和讨论的内容。自然关注可以降低对付费触达的依赖，但关注只是获客输入，不是学习或订阅健康度的证明。

团队需要把渠道曝光连接到合格访问、激活、持续学习行为、转化与品牌护栏。

### 3. 市场扩张不止是逐字翻译

国际化扩大潜在受众，但还涉及市场需求、本地渠道、文化解释、支付条件、设备限制和运营重点。

扩张要按市场评估。全球汇总增长可能掩盖个别地区的低留存或低商业化水平。

### 4. 平衡探索与优化

Cheng 的 Explore-Exploit 模型把成熟增长工作分成两个组合：

- **Exploit：**改善已知引导、Paywall、定价、信息与留存界面。
- **Explore：**测试新循环、产品、受众、商业化界面或习惯机制。

缺少探索时，实验体系可能只擅长产生局部小提升；缺少优化时，有潜力的发现无法成为稳定系统。

### 5. 分开衡量留存与召回

连续使用和沉默后重新回来是不同的行为。两者都有价值，但混在一起会让流失问题看起来更健康。

可分别衡量新增、留存、召回和流失用户，并按免费与付费状态、市场、获客来源和学习意图分群。

### 6. 让游戏化服务于学习

Cheng 把游戏化分为核心循环、长期 Meta Game 和用户 Profile 或身份。这些系统可以让进步可见并支持习惯。

护栏是学习价值。如果用户只优化积分却避开有意义的练习，参与度上升也没有完成产品任务。

### 7. 把 Streak 设计成留存系统

Jackson Shuttleworth 的叙述表明，Duolingo 的 Streak 不只是连续天数计数器。它组合清晰的每日状态、即时反馈、里程碑、情境化提醒、通过 Streak Freeze 提供的恢复机会，以及不断回到学习行为的循环。

这套系统也说明习惯机制为什么需要护栏。恢复应该保护长期投入，但不能让每日练习失去意义；通知应该促进回访，但不能造成疲劳；商业化也不应该利用用户对失去累计进度的恐惧。

Shuttleworth 讨论了超过 600 次 Streak 相关实验。这是参与者报告的迭代深度，不能证明实验数量或 Streak 单独造成了公司增长。

### 8. 连接可及性、动机与商业化

Luis von Ahn 围绕广泛提供教育机会理解 Duolingo 的系统。Freemium 创造覆盖，动机机制支持重复练习，订阅和广告为产品提供资金。这些层次会相互约束：激进商业化可能削弱可及性，不能改善练习的参与也没有完成教育任务。

公司还把产品人格延伸到品牌分发。猫头鹰、趣味提醒和平台原生内容在把用户带回产品时具有一致性。公共关注仍只是获客输入，不是学习、留存或订阅质量的证据。

AI 可能扩大内容和互动，但生成数量需要通过准确性、学习结果、信任和持续使用评估，而不能只看生产速度。

### 可以迁移的方法

1. 把订阅增长拆成获客、激活、转化、留存和收入质量。
2. 按市场、渠道、产品状态和用户意图划分指标。
3. 把品牌与社交触达视为关于合格获客的假设。
4. 为 Explore 与 Exploit 分别保留明确资源。
5. 分开报告留存、召回和流失用户。
6. 把每项游戏化机制映射到创造真实价值的行为。
7. 为学习质量、信任、通知疲劳和取消设置护栏。
8. 保持 Streak 规则清晰，提供有限恢复，并验证被计数行为能否预测持续价值。
9. 明确免费价值、付费价值、广告和服务成本如何相互支持，同时不破坏产品使命。

### 不应该直接复制的部分

- 没有相同品牌许可度和风险控制时，不要模仿 Duolingo 的表达语气。
- 没有基于价值的核心循环时，不要添加积分、连续记录或通知。
- 不要把社交触达当作订阅转化或留存证据。
- 不要未经调整就跨品类、账期使用消费订阅基准。

### 证据与局限

- 本案例结合不同时期与职能的参与者视角，上述顺序属于编辑整合。
- 这些来源没有提供完整内部实验、Cohort 定义、归因和订阅经济数据。
- 产品质量、学习效果、定价、市场时机、团队执行与品牌都会影响结果。
- 本案例不使用公司公开增长或收入数字证明单项方法的因果效果。
