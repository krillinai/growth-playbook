## 与增长相关的工作

Egner 对 Airtable 早期营销的描述，从理解高参与度用户开始：把他们的语言、使用场景和反馈转化成产品与分发决策。客户成功与营销紧密协作，因为用户对话既提供产品洞察，也提供可信的客户故事。

这套方法适合使用场景广泛的产品。灵活产品不能只依赖宽泛的品类表述，团队还要识别谁获得了显著价值、他们建立了什么，以及这些证据如何帮助下一位用户理解产品。

## 核心增长思想

### 1. 从超级用户开始

高参与度用户会暴露具体任务、语言、阻力和意外工作流。研究这些用户可以同时改善定位、新用户引导、模板和产品路线图。

### 2. 连接客户成功与营销

客户成功在真实情境中听到问题，营销把重复出现的模式转化成更清晰的教育与分发。更短的反馈路径能降低营销活动脱离产品现实的风险。

### 3. 有选择地使用品牌渠道

Egner 讨论了 Airtable 使用户外广告，让小公司在竞争市场中显得更可信、更有存在感。可迁移原则是渠道、信息与集中投入的匹配，而不是创业公司都应该投放户外广告。

### 4. 把模板当作产品教育

模板用一个初始状态和明确任务，让高度灵活的产品变得容易理解。它可以帮助发现与激活，但对收入的贡献仍取决于访问者意图和后续行为。

### 5. 避免没有增长假设的活动

PR、营销活动和内容都应该服务于明确受众、产品行为或战略目标。知名度本身不能证明激活、留存或商业结果。

## 运行摘要

```text
找到高参与度用户
→ 提取任务、语言与阻力
→ 反馈到产品与定位
→ 把成功工作流转化成故事和模板
→ 衡量采用与持续使用
```
