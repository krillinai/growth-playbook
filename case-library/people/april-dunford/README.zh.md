**可执行 Skill：**[April Dunford Positioning Lens](skill/)

适用于买家无法理解产品、销售线索质量不稳定、定位被简化成文案，或 AI 与产品组合主张高度同质化的情况。

1. 定义产品、客户分群、购买情境与本次定位决策。
2. 梳理竞争替代方案、独特能力、差异化价值、最佳客户，以及能让价值变得清晰的市场品类。
3. 用客户证据、资格判断、销售叙事、产品组合一致性与品类创建成本检验定位。

## 与增长相关的经历

Dunford 的公开方法来自她在 B2B 科技公司中的运营与咨询经验。她关注的典型问题是：产品本身有价值，但买家、销售团队或市场无法迅速理解，因为产品所处的竞争语境不清晰。

定位会影响获客质量、销售转化、定价能力、产品策略、发布和留存预期，但不能制造产品市场匹配，也不能替代产品证据。

## 核心定位思想

### 1. 定位是语境，不是口号

定位定义了一个能够让正确客户理解产品价值的市场语境。Messaging、品牌、Campaign 与销售故事是对定位的表达，不应该反过来承担发现定位的工作。

### 2. 从竞争替代方案开始

询问客户：如果这个产品不存在，他们会怎么办？答案可能是直接竞争对手、表格、内部流程、服务商、通用工具，或者什么都不做。这些才是影响客户价值判断的真实比较对象。

### 3. 把独特能力翻译为客户价值

先找出替代方案难以提供的能力，再把能力连接到客户在意的结果。没有后果的功能不是强差异化，“简单”“强大”等通用利益也无法形成有效定位。

```text
独特能力
-> 运营或使用结果
-> 客户与业务价值
-> 支撑证据
```

### 4. 从差异化价值定义最佳客户

目标客户不是所有可能使用产品的人，而是最重视差异化价值、具备获得该价值的条件，并且能理解现有替代方案不足的人群。

### 5. 选择能提供有效语境的市场品类

品类会告诉客户产品是什么、适合谁、应该和什么比较，以及如何评估。现有品类能解释价值时就使用它；只有当现有框架持续掩盖差异化价值，并且公司有能力教育市场时，才考虑重构或创建品类。

### 6. 按依赖顺序完成定位

```text
竞争替代方案
-> 独特属性
-> 差异化价值
-> 最佳客户
-> 市场品类
-> 强化价值的趋势
```

如果从品类、Persona 或口号开始，团队可能在理解客户选择原因之前就锁定错误假设。

### 7. 区分定位、战略与愿景

定位说明产品现在在哪个语境中能够取胜；战略决定公司在哪里以及如何竞争；愿景描述公司希望创造的未来。三者相互影响，但回答的是不同问题。

### 8. 把定位转化为销售叙事

销售 Pitch 应先帮助买家理解市场问题与评估标准，再展示产品功能。

```text
市场洞察
-> 当前替代方案及其限制
-> 理想未来
-> 差异化价值
-> 产品能力与证据
-> 异议、下一步与决策
```

### 9. 用定位指导资格判断与 Discovery

Qualification 应判断潜在客户是否具备使差异化价值成立的问题、条件、紧迫性与优先级。Discovery 不是通用问题清单，而是在帮助买家建立决策语境的同时验证匹配度。

### 10. 有意识地定位产品组合

单产品与多产品公司的定位决策不同。产品组合可以使用统一定位、独立产品定位，或者平台加产品的架构。选择取决于购买语境、共享价值、Cross-sell 逻辑，以及组合是否让差异更清楚。

### 11. 通过市场行为验证定位

定位是一项需要执行和证据的假设。验证目标客户是否理解语境、认可差异化价值、推进销售、接受价格、按预期原因采用并留存。不要把定位测试缩减为对口号的偏好调查。

### 12. AI 时代更需要语境与证据

当功能更容易复制、许多厂商都声称拥有 AI 能力时，客户更需要明确知道：为什么在这个场景中选择这个产品，而不是其他替代方案。工作流匹配、专有数据、信任、集成、结果与证据可能比通用的 AI 标签更重要。

## 定位系统总结

| 层级 | 诊断问题 | 证据 |
| --- | --- | --- |
| 替代方案 | 没有产品时客户会怎么办？ | Win/Loss、访谈、替换行为 |
| 能力 | 产品能够提供哪些替代方案没有的能力？ | 产品、技术与运营证据 |
| 价值 | 这些差异为什么重要？ | 收益、成本、速度、风险与质量 |
| 最佳客户 | 谁最重视并能获得这些价值？ | 分群转化、紧迫性、留存、支付意愿 |
| 品类 | 哪种语境最容易解释价值？ | 客户理解、评估标准、销售推进 |
| 叙事 | 应该如何解释购买决策？ | Discovery、异议处理、Pitch 转化 |
| 执行 | 定位是否贯穿整个客户旅程？ | 获客质量、销售周期、定价、采用与留存 |
