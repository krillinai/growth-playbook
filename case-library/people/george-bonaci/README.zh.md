## 核心增长思想

### 1. 管理增长押注组合

增长资源应覆盖可靠优化、相邻机会和高不确定性押注。组合可以保护学习空间，避免让所有想法在虚假精度下竞争。

### 2. 平衡速度与质量

当每个周期产生可信证据时，速度才有意义。薄弱埋点、模糊责任和样本不足只会创造快速活动，而不是快速学习。

### 3. 用复盘改善系统

复盘应解释假设、决策、结果与流程改变，而不是追责仪式。产出是更好的下一次决策，不只是一份文档。

### 4. 从产品、分发与运营寻找 Alpha

增长优势来自非显而易见的杠杆，而不是复制渠道。差异化产品价值、未充分利用的分发和运营能力都可能创造优势。

### 5. 把注意力视为稀缺资产

品牌、活动、实用物品和鲜明沟通可以获得无法被末次归因完整捕捉的注意力。衡量要组合实验、先行指标、客户证据与业务结果。

### 6. 围绕 AI Agent 重建工作流

Ramp 对 AI 营销的描述，是围绕共享 Agent 和内部系统重组重复研究、生产与运营工作。人的职责转向判断、语境、创意方向、验证和编排。
