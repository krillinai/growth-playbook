## Core Growth Ideas

### 1. Manage a Portfolio of Bets

Growth resources should span reliable optimizations, adjacent opportunities, and higher-uncertainty bets. A portfolio protects learning without making every idea compete on false precision.

### 2. Balance Velocity and Quality

Speed matters when each cycle produces trustworthy evidence. Weak instrumentation, ambiguous ownership, and underpowered tests create fast activity rather than fast learning.

### 3. Use Postmortems to Improve the System

Postmortems should explain assumptions, decisions, outcomes, and process changes without becoming blame rituals. The output is a better next decision, not merely a document.

### 4. Find Alpha Through Product, Distribution, and Operations

Bonaci frames growth advantage as finding non-obvious leverage rather than copying channels. Differentiated product value, underserved distribution, and operational capability can each create advantage.

### 5. Treat Attention as a Scarce Asset

Brand, events, useful objects, and distinctive communication can earn attention that cannot be fully captured by last-touch attribution. Measurement should combine experiments, leading indicators, customer evidence, and business outcomes.

### 6. Rebuild Workflows Around AI Agents

Ramp's described AI marketing work reorganizes repeated research, production, and operational tasks around shared agents and internal systems. Human roles shift toward judgment, context, creative direction, validation, and orchestration.
