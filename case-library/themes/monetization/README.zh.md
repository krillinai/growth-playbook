# 商业化

> 商业化定义谁为哪种价值付费、通过什么模式付费，以及经济性如何成立。收入是价值、定价、使用、留存与成本共同产生的结果。

**[English](README.md)｜[简体中文](README.zh.md)**

关于定价模型、单位经济性与诊断，参见主手册[商业化](../../../handbook/monetization/README.zh.md)。

## 公司证据

| 公司 | 价值与付款人 | 决策视角 | 边界 |
| --- | --- | --- | --- |
| [Ramp](../../companies/ramp/README.zh.md) | 企业为金融效率和节省付费 | 回本、贡献毛利、一致激励 | B2B 销售和品类经济性特殊 |
| [Duolingo](../../companies/duolingo/README.zh.md) | 学习者为便利付费，同时保持学习可及性 | Freemium、订阅、留存、广告权衡 | 使命、习惯和广告边界影响包装 |
| [Lovable](../../companies/lovable/README.zh.md) | 构建者为 AI 软件生产付费 | 用量经济性、套餐、信任、年付 | 模型成本与能力快速变化 |
| [字节跳动](../../companies/bytedance/README.zh.md) | 消费注意力与产品矩阵经济性 | 预测 LTV、多周期 ROI、优惠券 | 规模、数据与组合效应限制迁移 |
| [HubSpot](../../companies/hubspot/README.zh.md) | 销售专业人员与 Mid-Market 账户为降低不确定性和销售工作流价值付费 | Freemium、套餐定价、最低席位、年度合同与 Channel-Model Fit | 2014-2016 近似回顾；后续定价与结果不在范围内 |

## 对比与迁移

Ramp 强调可衡量企业节省，Duolingo 平衡可及性与订阅价值，Lovable 面对波动的 AI 交付成本，字节跳动跨产品组合使用预测经济性。迁移前需要明确谁使用、决策和付款，成本如何随使用变化，LTV 对留存与毛利有多敏感，以及套餐是否损害信任或采用。

## 证据与局限

- 公司证据主要来自参与者访谈和公开材料，不是完整内部因果评估。
- 公司规模、市场、产品频率、监管与数据能力限制直接迁移。
- 发布方公布的增长与收入数字不用于证明单项方法有效。
