# 激励增长：跨公司案例

> 激励增长使用现金、金币、优惠券、恢复、身份或权限改变行为。只有激励强化产品价值，并在成本、作弊和蚕食后创造持续增量时，它才有效。

关于增量、奖励经济性与激励后留存，参见[激励增长](../../../handbook/incentive-growth/README.zh.md)。

| 公司 | 证据与方法 |

## 对比与迁移

字节跳动组合金钱奖励、产品变体、产品互导与反作弊；Duolingo 围绕学习进度使用功能型和保护型奖励；Facebook 的邀请服务于网络价值。它们通过不同动机改变行为，风险结构也不同。

迁移前需要定义自然基线、Holdout、完整奖励成本、激励后留存、作弊、蚕食、公平性和剩余产品价值。

## 证据与局限

- 证据主要来自参与者访谈和公开材料，不是完整内部因果评估。
- 公司规模、品类、监管、数据与人才条件限制直接迁移。
- 发布方增长、收入、估值与客户数主张不用于证明单项方法。
