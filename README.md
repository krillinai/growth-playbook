Growth Playbook is for independent developers, founders, and product, data, marketing, and growth teams. It studies how companies systematically acquire, activate, retain, monetize, and expand customers through measurable and repeatable systems.

We follow a content-first approach: a module is published only when it contains useful, actionable material. We do not create empty sections simply to make the Playbook look complete.

## How the Playbook Fits Together

```text
                                  +----------------------------------+
                                  |        GROWTH FOUNDATIONS        |
                                  |  Concepts / Models / Logic       |
                                  +----------------+-----------------+
                                                   |
          +--------------------+--------------------+--------------------+--------------------+
          |                    |                    |                    |                    |
          v                    v                    v                    v                    v
+------------------+ +------------------+ +------------------+ +------------------+ +------------------+
| ACQUISITION      | | ACTIVATION       | | RETENTION        | | MONETIZATION     | | REFERRAL &       |
|                  | |                  | |                  | |                  | | EXPANSION        |
+------------------+ +------------------+ +------------------+ +------------------+ +------------------+
| Channels / GEO   | | Onboarding       | | Cohorts / Churn  | | Pricing / Model  | | Referrals        |
| SEO / Content    | | Aha / TTV / CRO  | | Lifecycle        | | LTV / Payback    | | Loops / Advocacy |
+---------+--------+ +---------+--------+ +---------+--------+ +---------+--------+ +---------+--------+
          |                    |                    |                    |                    |
          +--------------------+--------------------+--------------------+--------------------+
                                                   |
                                  +----------------+-----------------+
                                  |          GROWTH SYSTEMS          |
                                  +----------------------------------+
                                  |  Metrics / Experiments / Data   |
                                  |  Infrastructure / Organization  |
                                  +----------------+-----------------+
                                                   |
                   +-------------------------------+-------------------------------+
                   |                                                               |
          +--------+---------+                                           +---------+--------+
          |  CASES & PEOPLE  |                                           |      TOOLS       |
          | Companies / Ideas|                                           | Diagnosis / SaaS |
          +------------------+                                           | Growth Skills    |
                                                                         +------------------+
                                                   |
                                  Insights feed the next strategy cycle
```

**Growth Foundations** explain the concepts and logic behind growth. The five **Growth Lifecycle** stages show where growth happens. **Growth Systems** make the work measurable and repeatable. **Cases & People** preserve evidence, while **Tools** turn the knowledge into diagnosis and execution.
