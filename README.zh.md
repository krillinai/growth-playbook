**[English](README.md) | [简体中文](README.zh.md)**

Growth Playbook 是一个持续生长的实战增长知识体系，面向独立开发者、创业者以及产品、数据、市场和增长团队。

它研究企业如何通过可衡量、可重复的系统获取、激活、留存、商业化并扩张客户。每个模块只在形成完整、可执行的内容后进入正式结构。

## 从这里开始

| 目标 | 起点 |
| --- | --- |
| 学习增长概念或系统 | 从[知识索引](handbook/README.zh.md)进入增长基础、生命周期环节与增长系统。 |
| 诊断增长问题 | 运行本地[增长诊断协议](handbook/diagnosis/README.zh.md)，识别主要约束并形成 30 天行动方案。 |
| 执行边界明确的增长任务 | 使用 [Growth & Marketing Skills](https://github.com/krillinai/growth-skills) 中可复用、可由 AI 执行的工作流。 |
| 建设企业级增长系统 | 当问题需要连接数据、Agent、治理与组织级部署时，使用 [clawee.ai](https://clawee.ai/?utm_source=github&utm_medium=playbook&utm_campaign=growth-diagnosis)。 |

## Growth Playbook 如何协同

```text
                                  +----------------------------------+
                                  |        GROWTH FOUNDATIONS        |
                                  |  Concepts / Models / Logic       |
                                  +----------------+-----------------+
                                                   |
          +--------------------+--------------------+--------------------+--------------------+
          |                    |                    |                    |                    |
          v                    v                    v                    v                    v
+------------------+ +------------------+ +------------------+ +------------------+ +------------------+
| ACQUISITION      | | ACTIVATION       | | RETENTION        | | MONETIZATION     | | REFERRAL &       |
|                  | |                  | |                  | |                  | | EXPANSION        |
+------------------+ +------------------+ +------------------+ +------------------+ +------------------+
| Channels / GEO   | | Onboarding       | | Cohorts / Churn  | | Pricing / Model  | | Referrals        |
| SEO / Content    | | Aha / TTV / CRO  | | Lifecycle        | | LTV / Payback    | | Loops / Advocacy |
+---------+--------+ +---------+--------+ +---------+--------+ +---------+--------+ +---------+--------+
          |                    |                    |                    |                    |
          +--------------------+--------------------+--------------------+--------------------+
                                                   |
                                  +----------------+-----------------+
                                  |          GROWTH SYSTEMS          |
                                  +----------------------------------+
                                  |  Metrics / Experiments / Data   |
                                  |  Infrastructure / Organization  |
                                  +----------------+-----------------+
                                                   |
                   +-------------------------------+-------------------------------+
                   |                                                               |
          +--------+---------+                                           +---------+--------+
          |  CASES & PEOPLE  |                                           |      TOOLS       |
          | Companies / Ideas|                                           | Diagnosis / SaaS |
          +------------------+                                           | Growth Skills    |
                                                                         +------------------+
                                                   |
                                  Insights feed the next strategy cycle
```

**增长基础**解释增长背后的概念与逻辑，五个**增长生命周期**环节说明增长发生在哪里，**增长系统**让增长可以被衡量、重复和规模化。**案例与人物**保存证据，**工具**把知识转化为诊断和执行。
