**[English](README.md) | [简体中文](README.zh.md)**

Growth Playbook 是一个持续生长的实战增长知识体系，面向独立开发者、创业者以及产品、市场和增长团队。

它不会先搭建一套庞大但空洞的框架。每个模块都会在形成完整、可执行的内容后发布，逐步覆盖从策略制定到内容生产、营销分发和数据优化的完整增长闭环。

## Growth Playbook 如何协同

```text
                                  +----------------------------------+
                                  |        GROWTH FOUNDATIONS        |
                                  |  Concepts / Models / Logic       |
                                  +----------------+-----------------+
                                                   |
          +--------------------+--------------------+--------------------+--------------------+

+------------------+ +------------------+ +------------------+ +------------------+ +------------------+

| SEO / Content    | | Aha / TTV / CRO  | | Lifecycle        | | LTV / Payback    | | Loops / Advocacy |
+---------+--------+ +---------+--------+ +---------+--------+ +---------+--------+ +---------+--------+
          |                    |                    |                    |                    |
          +--------------------+--------------------+--------------------+--------------------+
                                                   |
                                  +----------------+-----------------+
                                  |          GROWTH SYSTEMS          |
                                  +----------------------------------+
                                  |  Metrics / Experiments / Data   |
                                  |  Infrastructure / Organization  |
                                  +----------------+-----------------+
                                                   |
                   +-------------------------------+-------------------------------+
                   |                                                               |
          +--------+---------+                                           +---------+--------+
          |  CASES & PEOPLE  |                                           |      TOOLS       |
          | Companies / Ideas|                                           | Diagnosis / SaaS |
          +------------------+                                           | Growth Skills    |
                                                                         +------------------+
                                                   |
                                  Insights feed the next strategy cycle
```

**增长基础**解释增长背后的概念与逻辑，五个**增长生命周期**环节说明增长发生在哪里，**增长系统**让增长可以被衡量、重复和规模化。**案例与人物**保存证据，**工具**把知识转化为诊断和执行。
