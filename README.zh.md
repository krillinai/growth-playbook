Growth & Marketing Playbook 是一本持续生长的实战增长营销手册，面向独立开发者、创业者以及产品、市场和增长团队。

它不会先搭建一套庞大但空洞的框架。每个模块都会在形成完整、可执行的内容后发布，逐步覆盖从策略制定到内容生产、营销分发和数据优化的完整增长闭环。

## 先诊断，再执行

**诊断增长约束，参考真实案例，使用开源 Skills 执行。**

[增长诊断协议](handbook/diagnosis/README.zh.md)把可见症状转化为一个核心约束、证据账本、30 天行动方案与正确的执行路径。

| 诊断路径 | 入口 |
| --- | --- |
| 获客 | [诊断需求、渠道与获客经济性](handbook/diagnosis/README.zh.md#获客诊断) |
| 激活 | [诊断首次价值路径](handbook/diagnosis/README.zh.md#激活诊断) |
| 留存 | [诊断重复价值与流失](handbook/diagnosis/README.zh.md#留存诊断) |
| 商业化 | [诊断可持续收入转化](handbook/diagnosis/README.zh.md#商业化诊断) |
| 增长循环 | [诊断复利式分发与价值](handbook/diagnosis/README.zh.md#增长循环诊断) |
| 增长系统 | [诊断衡量、决策与执行系统](handbook/diagnosis/README.zh.md#增长系统诊断) |

使用 Playbook 完成诊断并理解方法，使用 [krillinai/growth-skills](https://github.com/krillinai/growth-skills) 执行边界明确的开源工作流；当实施需要持续系统连接、私有数据、治理或跨团队协同时，使用 [clawee.ai](https://clawee.ai/)。

## 手册体系如何协同

```text
                                  +----------------------------------+
                                  |       GROWTH DIAGNOSIS           |
                                  |  Constraint / Evidence / Action  |
                                  +----------------+-----------------+
                                                   |
                                  +----------------------------------+
                                  |        GROWTH FOUNDATIONS        |
                                  |  Growth Metrics                  |
                                  |  Product-Market Fit              |
                                  +----------------+-----------------+
                                                   |
          +--------------------+--------------------+--------------------+--------------------+
          |                    |                    |                    |                    |
          v                    v                    v                    v                    v
+------------------+ +------------------+ +------------------+ +------------------+ +------------------+
| ACQUISITION      | | ACTIVATION &     | | RETENTION &      | | MONETIZATION     | | REFERRAL &       |
|                  | | CONVERSION       | | EXPANSION        | |                  | | GROWTH LOOPS     |
+------------------+ +------------------+ +------------------+ +------------------+ +------------------+
| Acquisition      | | Activation       | | Retention        | | Monetization     | | Referrals        |
| GEO              | | Funnels / CRO    | | Lifecycle        | | Pricing/Models   | | Growth Loops     |
| Content/Social   | | Onboard / TTV    | | Lifecycle Mktg   | | LTV / Payback    | | Virality         |
| Paid             | |                  | | Churn Prevent.   | | Profit Growth    | | Product Loops    |
|                  | |                  | |                  | |                  | | Advocacy         |
+---------+--------+ +---------+--------+ +---------+--------+ +---------+--------+ +---------+--------+
          |                    |                    |                    |                    |
          +--------------------+--------------------+--------------------+--------------------+
                                                   |
                                  +----------------+-----------------+
                                  |          GROWTH SYSTEMS          |
                                  +----------------------------------+
                                  |  Growth Capabilities            |
                                  +----------------+-----------------+
                                                   |
                                  +----------------+-----------------+
                                  |   REFERENCE & EXECUTION LAYERS   |
                                  +----------------------------------+
                                  |  Case Library                   |
                                  |  Growth Stages / Capabilities    |
                                  |  Companies / People & Ideas      |
                                  +----------------------------------+
                                  |  Toolkit                        |
                                  |  Tools / Growth Skills           |
                                  +----------------------------------+
                                                   |
                                  Insights feed the next strategy cycle
```

**增长诊断**识别当前约束与下一项决策，**增长基础**定义如何理解增长，**增长旅程**说明增长发生在哪里，**增长系统**说明增长如何被组织和执行。**Case Library（案例资料库）**沉淀人物与公司证据，**Toolkit（工具箱）**连接工具与 Growth Skills。
