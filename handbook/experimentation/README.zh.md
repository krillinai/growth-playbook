# 实验系统

> 实验是降低重要不确定性的决策系统，而不是追求测试数量的竞赛。

**[English](README.md)｜[简体中文](README.zh.md)**

## 决策模型

```text
决策 -> 不确定性 -> 假设 -> 干预 -> 证据 -> 行动 -> 可复用学习
```

上线前定义人群、处理、核心结果、护栏、最小有意义效应、周期，以及正向、负向和无结论结果各自触发的行动。

## 指标体系

| 指标 | 含义 |
| --- | --- |
| 效应量 | 处理造成或相关的差异 |
| 置信区间 | 与观察数据相容的效应范围 |
| 统计功效 | 发现目标效应的概率 |
| 结论率 | 能支持计划决策的实验 / 已完成实验 |
| 决策周期 | 从不确定性到实施决策的时间 |
| 学习复用 | 后续决策使用已记录证据的程度 |

## 问题诊断

| 模式 | 下一步检查 |
| --- | --- |
| 实验多，结果变化小 | 是否只测试低影响表面，或策略薄弱 |
| 大多无结论 | 功效、埋点、曝光与效应阈值 |
| 主指标正向，护栏受损 | 局部优化或新奇效应 |
| 结果很少落地 | 负责人、决策权与实施成本 |

## 操作方法

1. 优先确定决策和信息价值。
2. 预先定义假设、设计、指标、阈值、护栏和行动规则。
3. 验证分组、曝光、埋点与样本比例。
4. 读取效应量与不确定性，而不是只看显著性。
5. 记录解释、决策、局限与后续行动。
6. 产品、人群或模型变化时重新审视结果。

## 常见错误

- 优化实验数量；反复查看并提前停止；多指标不校正；把不显著当成无效；把结果外推到未测试人群。
