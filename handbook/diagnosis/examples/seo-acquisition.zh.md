# 模拟 SEO 获客诊断

**[English](seo-acquisition.md) | [简体中文](seo-acquisition.zh.md) | [JSON（JavaScript 对象表示法）](seo-acquisition.json)**

> 这是一个针对未具名 B2B（企业对企业）工作流自动化产品的模拟诊断。所有公司信息、日期和指标均为虚构，且未归因于所引用案例。

## 1. 诊断范围

| 字段 | 内容 |
| --- | --- |
| 产品 | 连接审批、交接和报告工作流的 B2B 工作流自动化 SaaS（软件即服务）；预期每周使用 |
| 客户 | 用户：运营经理和流程负责人；付费者：运营或 IT 负责人；优先分群：正在评估工作流自动化的 100-1,000 人公司；当前替代方案：电子表格、邮件和通用项目工具 |
| 商业模式 | 按工作区和活跃用户计价的年度订阅；销售辅助购买流程；毛利主要受实施和支持工作影响 |

| 决策窗口 | 132 天，从 2026-07-21 至 2026-11-30 最终决策 |

## 2. 目标结果

## 3. 主要约束与替代约束

| 字段 | 内容 |
| --- | --- |
| 主要约束 | 自然增长集中在信息型查询和页面，其访客很少表现商业意图，也很少形成留存且销售合格的需求 |
| 诊断领域 | `acquisition` |
| 机制 | 新增可见性主要来自宽泛的教育型查询。这些会话进入的问题到产品衔接薄弱的文章，高意图行动转化明显更低；注册后激活和留存也更弱；销售接受商机创建率只有商业/问题感知会话的九分之一。技术可发现性限制了部分高意图页面，但已经获得商业流量的可索引页面展现出明显更强的下游质量。 |
| 选择理由 | 只有因为查询结构与下游证据一致，才选择意图错配：信息型会话占流量 69%，到 Demo/定价/合格联系的转化为 0.8%，商业页面为 4.6%；合格 Pipeline 创建率为 0.12%，商业页面为 1.08%；其注册账户 30 天留存激活为 18%，商业意图账户为 41%。结论仍是观察性的；10% 查询未分类，辅助转化不完整，技术问题也影响高意图覆盖。因此置信度为中等而非高。 |
| 置信度 | 0.67 |

### 替代约束

| 替代约束 | 未选为主要约束的原因 | 置信度 |
| --- | --- | ---: |
| 技术可发现性压制商业页面 | 有边界的抓取发现，50 个高意图抽样页面中有 14 个存在 canonical、内链或渲染问题，因此影响显著。但技术健康的商业页面已显示远高于信息页面的合格 Pipeline 创建率；仅修复技术问题不会让宽泛的信息流量变合格。 | 0.52 |
| 落地页转化和产品信息薄弱 | 商业页面到高意图行动的转化仍只有 4.6%，存在可达提升空间。但其商机率和留存激活率显著强于信息页面，因此转化摩擦次于新增流量来自何种意图。 | 0.49 |
| 归因遗漏自然搜索辅助 Pipeline | 当前路径模型把 22 个辅助商机加到 68 个来源商机上，得到当前每月 90 个影响商机，反驳严格末次触点解读。由于没有可比的此前周期辅助商机基线，无法建立影响 Pipeline 趋势。归因少计可能解释部分差距，但在缺少该基线、更完整身份覆盖和增量证据时，不能推翻查询意图与下游质量证据。 | 0.45 |
| 销售资格判断或跟进恶化 | 潜客响应时间中位数和接受标准稳定，商业意图 Cohort 的商机率也保持稳定。账户级路由缺陷仍可能存在，因为 17% 可识别自然联系缺少可靠的首次会话关联。 | 0.28 |

### 缺失证据

- 39% 已注册自然账户和 37% 商机缺少查询到账户身份关联；在重分配大部分内容能力前，需要它确认留存激活和辅助 Pipeline 差异。
- 我们缺少可比的此前周期辅助商机基线和增量实验；需要它们建立影响 Pipeline 趋势，并判断当前 68 个来源商机加 22 个辅助商机、即 90 个影响商机中有多少应归功于自然搜索，然后才能改变诊断或据此扩大 SEO 投资。
- 大多数单个页面模板缺少成熟合格商机数；需要它区分查询意图、页面信息、转化设计和销售跟进。
- 我们缺少全站完整渲染抓取和日志视图；有边界审计足以确定抽样高意图页面优先级，但不能声称技术可发现性已被完整测量。
- 我们缺少按购买意图稳定关联的 AI 答案引用与引荐数据；GEO（生成式引擎优化）与可发现性相关，但在这些访问能连接到合格下游结果之前，不应改变当前诊断。
