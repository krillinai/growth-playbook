# 模拟消费产品留存诊断

**[English](consumer-retention.md) | [简体中文](consumer-retention.zh.md) | [JSON（JavaScript 对象表示法）](consumer-retention.json)**

> 这是一个面向未具名消费级膳食规划产品的模拟诊断。所有产品信息、日期和数值均为虚构数据，不归因于引用案例。

## 1. 诊断范围

| 字段 | 内容 |
| --- | --- |
| 产品 | 生成每周膳食计划和购物清单的消费级应用；自然使用周期为每周 |
| 客户 | 主要用户和付费者是为家庭规划膳食的英语市场成年人；优先分群：通过直接访问、自然流量、推荐和付费渠道获得的新免费及试用用户 |
| 商业模式 | 提供月度和年度高级套餐的 Freemium 消费订阅；价值以完成每周膳食计划和购物清单衡量 |
| 市场 | 英语市场的消费级膳食规划品类 |
| 阶段 | `scaling` |
| 诊断领域 | `retention` |
| 决策窗口 | 52 天，从 2026-07-21 至最早 2026-09-11 的最终决策 |

## 3. 主要约束与替代约束

| 字段 | 内容 |
| --- | --- |
| 主要约束 | 应用无法再为许多新用户提供足够相关且可靠的每周计划，导致重复的膳食规划价值在第 4 周前消失 |
| 诊断领域 | `retention` |
| 机制 | 快速的目录和排序改动增加了不可用、重复和不相关的菜谱结果，搜索及清单同步失败也在上升。用户在第 1 周完成的有用计划减少，之后几周遇到的新鲜度降低、修正工作增加，因此不再按每周自然周期返回。更高的通知曝光随后只会提醒用户薄弱的价值，加速关闭通知，而不能恢复价值。 |
| 选择理由 | 使用与头部指标相同的全部合格用户分母，把此前 45% 自然/推荐、40% 核心付费、6% 达人短视频和 9% 未知来源权重应用于此前 35%、31%、20% 和 25% 的来源率，得到 31.6%。把当前 30%、25%、36% 和 9% 的权重应用于当前 29%、25%、13% 和 20% 的来源率，得到 21.4%，总降幅为 10.2 个百分点。把此前权重应用于当前率得到 25.6%，因此结构解释 4.2 个百分点，来源内部或产品效应解释剩余 6.0 个百分点。各来源中，第 1 周计划和清单完成都能预测第 4 周价值，同时完成率、内容质量、搜索可靠性和同步可靠性均恶化。 |
| 置信度 | 0.72 |

### 替代约束

| 替代约束 | 未选为主要约束的原因 | 置信度 |
| --- | --- | ---: |
| 达人短视频扩张后的获客质量下降 | 达人短视频用户留存更差，可审计的结构变化解释 4.2 个百分点；但用当前来源率和此前结构估算的留存仍只有 25.6%，低于此前 31.6%。获客质量影响显著，却不能解释更大的来源内部或产品损失。 | 0.61 |
| 通知疲劳导致用户退出 | Push 关闭率上升，收到至少五条 Push 的用户留存更差；但通知频率部分由不活跃和计划完成不佳触发。预设的恢复价值对比通知上限检验，将判断恢复价值是否优于减少提醒。 | 0.48 |
| 产品可靠性失败是主要原因 | 崩溃、搜索和清单同步失败与低留存相关，但检测到的严重失败影响 28% 的新用户，不能单独解释全部降幅。可靠性被纳入恢复重复价值的方案，并单独报告。 | 0.56 |
| 季节性降低了膳食规划需求 | 对比相同来源和家庭意图分群后，降幅仍然存在，而且每周购物清单创建在季节性假期之前就已下降。由于没有去年同期的匹配 Cohort，季节性仍未排除。 | 0.24 |

### 缺失证据

- 我们缺少区分恢复重复价值与减少通知曝光的随机证据；在为默认新用户体验选择产品质量改动、生命周期改动或两者并用之前，需要补足该证据。
- 我们缺少达人短视频用户的获客增量 Holdout 和稳定的素材级意图标签；在减少支出或设置按留存价值调整的渠道预算之前，需要补足该证据。
- 我们缺少去年同期匹配的每周 Cohort，以及可靠的旅行或日程变化信号；需要用它们判断季节性是否会实质改变主要诊断。
- 我们缺少完整的服务端清单同步失败覆盖和经过验证的菜谱相关性评分；需要用它们估算重复价值损失中可靠性与内容相关性各占多少。
