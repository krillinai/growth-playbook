# 定位

> 定位定义了一个竞争语境，让最佳客户能够清楚理解产品的差异化价值。

**[English](README.md)｜[简体中文](README.zh.md)**

## 定位与增长

定位位于产品市场匹配与增长执行之间，决定应该验证哪些客户、替代方案、价值、品类、信息、渠道、销售方式与定价假设。

```text
产品与客户证据
-> 定位
-> Messaging 与销售叙事
-> 获客与客户资格判断
-> 采用、商业化与留存证据
-> 修正定位
```

定位不能修复薄弱的产品价值，但可以通过明确客户、场景和预期价值暴露问题。

## 定位五要素

### 竞争替代方案

识别没有产品时客户会使用什么或做什么，包括直接竞争对手、内部工具、人工流程、服务商、通用软件和不采取行动。

### 独特能力

找出替代方案在相关条件下无法提供的能力，并要求产品、技术、运营、数据、工作流、生态、信任或服务证据。

### 差异化价值

```text
能力 -> 后果 -> 客户价值 -> 证据
```

价值可能体现为收入、成本、时间、风险、质量、可达性、控制权或战略灵活性。避免使用所有竞争对手都能声称的利益。

### 最佳客户

定义最重视差异化价值，并具备获得该价值所需条件的客户。考虑紧迫性、使用场景、当前替代方案、工作流、公司状态、限制、购买流程、支付意愿、采用和留存。

### 市场品类

选择能够帮助买家理解产品是什么、如何评估以及哪些替代方案重要的语境。创建新品类需要高昂的市场教育成本，不应该成为默认选项。

## 定位顺序

1. 竞争替代方案
2. 独特能力
3. 差异化价值
4. 最佳客户
5. 市场品类
6. 相关趋势

趋势只能在价值明确后用于解释紧迫性或相关性，不能代替定位本身。

## 定位证据账本

| 主张 | 有效证据 | 常见错误 |
| --- | --- | --- |
| 客户把我们与 X 比较 | Win/Loss、销售对话、替换行为 | 使用内部竞争对手列表 |
| 能力 Y 独特 | 产品与运营证据 | 把 Roadmap 当成当前事实 |
| 差异 Y 创造价值 Z | 结果数据与客户语言 | 停留在功能或通用利益 |
| 分群 A 最匹配 | 转化、紧迫性、留存、支付意愿 | 定义最大的潜在市场 |
| 品类 C 能解释价值 | 理解度和评估行为 | 选择最流行的品类 |

将关键主张标记为 `verified`、`inferred` 或 `missing`。

## 定位工作坊

让产品、销售、市场、客户成功和管理层中掌握不同证据的人参与。会前确定产品、客户分群、地区、进入市场方式和决策周期。

工作坊应产出决策与待验证问题，而不是精美口号。为有争议的替代方案、能力、价值、分群和品类分配验证负责人。

## 销售叙事

1. 市场洞察或变化
2. 当前替代方案及其限制
3. 理想未来状态
4. 差异化价值
5. 产品能力与证据
6. 异议、决策标准与下一步

叙事应该帮助买家理解评估标准，不能用故事掩盖薄弱证据。

## 客户资格判断与 Discovery

验证潜在客户是否：

- 具有相关问题和当前替代方案；
- 面临产品所解决的限制；
- 重视差异化结果；
- 具备所需的工作流、权限、紧迫性、预算与准备度；
- 能够在可接受的实施风险下获得价值。

衡量合格客户推进、Win/Loss、销售周期、价格接受度、实施、激活和留存。

## 单产品与产品组合定位

购买语境一致时使用统一产品定位。产品组合可以采用独立产品定位、公司统一定位加产品价值、平台加模块，或基于共享能力的分群定位。

如果统一定位变得空泛，或者迫使客户理解不需要的能力，就不应强行合并。

## 测试与执行

| 环节 | 信号 |
| --- | --- |
| 认知 | 正确受众与问题识别 |
| 获客 | 合格响应与渠道信息匹配 |
| 销售 | Discovery、异议、推进、Win/Loss |
| 商业化 | 价格接受、包装匹配、折扣压力 |
| 采用 | 预期使用场景与 Time to Value |
| 留存 | 目标分群持续获得价值 |

信息测试是辅助证据，不能替代真实市场行为。

## AI 产品定位

“AI-powered”通常不足以形成差异。评估专有数据、工作流集成、输出质量、可靠性、控制权、安全、合规、切换成本、生态、服务和可验证结果。明确 AI 产品替代什么，以及目标客户为什么应该信任它。

## 指标

- 最佳客户在合格 Pipeline 中的占比；
- 合格到 Opportunity、Opportunity 到 Win 的转化；
- 各分群销售周期与折扣率；
- Win/Loss 原因和被替代方案；
- 价值信息理解度；
- 按承诺场景划分的激活与留存；
- 扩张与支付意愿；
- 与定位相关的异议频率。

## 护栏

- 不制造缺少证据的差异化。
- 不通过重新定义品类回避真实竞争。
- 不把市场趋势当成客户价值。
- 不利用定位吸引产品无法服务的客户。
- 没有实验或可信比较时，不声称定位造成结果。
