# Incentive Growth

> Incentive growth uses monetary, functional, social, or protective rewards to change behavior. It is sustainable only when retained incremental value exceeds cost, fraud, and cannibalization.

**[English](README.md)｜[简体中文](README.zh.md)**

## Incentive Model

```text
Eligible segment -> target behavior -> reward -> immediate lift
                 -> product value -> post-incentive behavior
                 -> cost, fraud, cannibalization, trust
```

Rewards can be cash, credit, coupons, access, status, progress protection, convenience, or social recognition. The reward must fit the behavior and user motivation.

## Metric System

| Metric | Definition or formula | Decision |
| --- | --- | --- |
| Eligible-to-action rate | `users taking target action / eligible users` | Immediate behavior change |
| Incremental lift | `treatment action rate - holdout action rate` | Behavior caused by the program |
| Cost per incremental action | `total reward and operating cost / incremental actions` | Efficiency |
| Post-incentive retention | Retained target behavior after reward ends | Durability |
| Fraud loss rate | `invalid reward cost / total reward cost` | Abuse exposure |
| Cannibalization | Reward paid for behavior that would occur organically | Displaced organic value |

## Diagnosis

| Pattern | Check |
| --- | --- |
| High lift, weak post-incentive retention | Reward-seeking without product value |
| Strong gross response, weak incrementality | Organic users receiving unnecessary rewards |
| Cost rises faster than action | Bid-up, repeated eligibility, diminishing response |
| New segment grows, existing segment falls | Portfolio or channel cannibalization |
| Complaints or gaming increase | Eligibility fairness, clarity, abuse controls |

## Operating Method

1. Define segment, target behavior, value hypothesis, eligibility, reward, duration, budget, and stop condition.
2. Use a randomized or credible holdout to estimate incrementality.
3. Separate rewarded behavior from organic product signals.
4. Measure post-incentive retention over the natural product cycle.
5. Monitor fraud, cannibalization, margin, trust, and operational burden.
6. Reduce or stop rewards when incremental value no longer exceeds full cost.

## Common Mistakes

- Rewarding a behavior with no durable customer value.
- Reading gross actions as incremental lift.
- Launching cash rewards without real-time risk controls.
- Making permanent economics depend on temporary subsidies.
